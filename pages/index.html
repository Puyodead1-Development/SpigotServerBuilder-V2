Welcome!

SpigotServerBuilder is a simple, easy-to-use, auto-updating tool for creating a Spigot server.

SpigotServerBuilders Features

  • Auto-Updating system
  • Sentry error tracking (can be turned off)
  • Downloading BuildTools if needed or use existing one from previous downloads.
  • Compiles Spigot
  • Moves Spigot JAR to a new folder
  • Creates EULA.txt and accepts it
  • Creates a launcher.bat for easy start
  • List of all Spigot versions
  • Easy to toggle options to fine tune your server build
